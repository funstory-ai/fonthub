Fonthub
===

A font has the following metadata:

- name
- license
- version
- categories
- widths
- weights
- styles
- languages
- source
- fontsFiles

You can search for fonts by the following criteria:

- widths
- weights
- styles
- languages

## API Examples

### Get all fonts

```
GET /api/fonts
```

### Get fonts by selector

```
GET /api/fonts/selector?width=full-width&weight=regular&style=normal&language=zh-CN
```
